# Aprimoramento da Interface de uma Trena Eletrônica com Machine Learning

Este projeto apresenta uma proposta de inovação para trenas eletrônicas, focada no aprimoramento da Experiência do Usuário (UX) através da integração de Machine Learning para a digitalização de medições manuscritas.

## 📝 Resumo do Projeto
[cite_start]O objetivo principal é solucionar a dificuldade enfrentada por profissionais de Engenharia e Arquitetura na extração manual de dados de medição para relatórios. [cite_start]A solução proposta utiliza o algoritmo **SVM (Support Vector Machine)** para reconhecer medidas escritas à mão em papel e transformá-las automaticamente em dados digitais dentro do sistema do dispositivo[cite: 3].

## 🚀 Principais Funcionalidades
- [cite_start]**Reconhecimento de Caracteres (OCR):** Transformação de medições manuscritas em dados digitais[cite: 3].
- [cite_start]**Geração de Relatórios:** Automatização da criação de relatórios de medição diretamente pelo sistema do equipamento[cite: 14].
- [cite_start]**Interface Otimizada:** Redução de 24% no tempo de medição em comparação ao uso de trenas convencionais[cite: 11].

## 🛠️ Tecnologias e Frameworks Utilizados
- [cite_start]**Linguagem:** Python [cite: 4]
- [cite_start]**Machine Learning:** Support Vector Machine (SVM) 
- [cite_start]**Dataset:** MNIST (60.000 exemplos de dígitos manuscritos) 
- [cite_start]**Design de UX/UI:** Figma (Prototipagem de interface) 
- [cite_start]**Metodologia de Design:** Framework PACT (Pessoas, Atividades, Contexto e Tecnologias) e ML-Process Canvas [cite: 15, 16]

## 📊 Resultados Obtidos
[cite_start]A simulação do modelo de Machine Learning apresentou uma **acurácia de 93,49%** nos dados de treinamento, demonstrando a viabilidade técnica da aplicação para o reconhecimento de dígitos em ambientes profissionais[cite: 4, 19].
